## Description

AgentDoom turns natural language into working software — instantly.

Type what you want ("a tip splitter for group dinners"), watch AI assemble it from pre-built components in seconds, and get a deployed web app with its own URL. No code, no config, no terminal.

But AgentDoom isn't just a builder. It's a social platform. Every tool you create appears in a scrollable feed. Others can browse, use, and Remix your tools — modifying them with plain English to create their own versions.

**What makes it different:**
- Sub-10-second generation from natural language — powered by 50+ battle-tested component primitives, not arbitrary code generation
- 97% first-attempt success rate (most AI code gen tools: ~60%)
- TikTok-style feed of live, running tools — scroll, use, fork directly from the feed
- Remix any tool in the feed with a single sentence. Consumer to creator in under a minute.
- Every tool gets its own shareable URL: agentdoom.ai/t/your-tool
- Mobile-first, works on any device, no sign-in required to browse

**What you can build today:** invoice generators, habit trackers, quiz builders, expense splitters, bio generators, pomodoro timers, content calendars, pricing calculators, form builders, rate calculators, onboarding checklists — any single-screen tool you can describe in plain English.

**Coming soon:** Marketplace (set a price, earn when someone buys, earn again when they remix your paid tool), Pro plan with custom domains and analytics, remix royalties.

## Maker Comment
*(Post within 60 seconds of going live)*

Hey PH —

We built AgentDoom because we think AI code generation is solving the wrong problem.

Every tool in this space competes on generation quality — but that improves automatically as models get better. The real problem is distribution. How do normal people find useful AI-built tools? How do they share and build on each other's work?

Our answer: make building software feel like scrolling TikTok.

Under the hood we use a component primitive architecture — 50+ pre-built, battle-tested React components that AI configures and wires together. No arbitrary code generation. This is why we hit 97% first-attempt success and sub-10-second deploys consistently, where most code gen tools hover around 60%.

The mechanic that surprised us most: remix chains. Someone builds a habit tracker. Three people fork it. One fork gets forked again. In 48 hours, one original tool becomes 12 distinct products — and the original creator's work keeps compounding without them touching it. That's the marketplace flywheel we're building toward.

We launched today after 4,200+ people joined the waitlist. Would love to know: what would you build first?

Happy to go deep on any technical questions below.
