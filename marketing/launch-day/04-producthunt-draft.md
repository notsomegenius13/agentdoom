## Tagline

Describe a tool in plain English. Get a live web app in 10 seconds. Share and remix in a TikTok-style feed.

## Description

AgentDoom turns natural language into working software — instantly.

Type what you want ("a tip splitter for group dinners"), watch AI assemble it from pre-built components in seconds, and get a deployed web app with its own URL. No code, no config, no terminal.

But AgentDoom isn't just a builder. It's a social platform. Every tool you create appears in a scrollable feed. Others can browse, use, and "Remix" your tools — modifying them with plain English to create their own versions.

**Key features:**
- Sub-10-second generation from natural language prompts
- 50+ component primitives: forms, calculators, trackers, charts, commerce widgets
- One-tap deploy with a shareable URL (agentdoom.ai/t/your-tool)
- TikTok-style feed of live, running tools
- Remix flow: fork any tool and modify with plain English
- Rich link previews for sharing on social media
- Mobile-first, works on any device

**Coming soon:** Marketplace for selling tools, Pro plan with custom domains and analytics, native iOS/Android app, remix royalties.

## Maker Comment

Hey PH! We built AgentDoom because we think AI code generation is solving the wrong problem.

Every tool in this space competes on generation quality — but that improves automatically as AI models get better. The real challenge is distribution. How do normal people FIND useful AI-built tools? How do they share and build on each other's work?

Our answer: make building software feel like scrolling TikTok. A feed of live tools. One-tap remix. Instant deploy. The viral loop handles distribution.

Under the hood, we use a component primitive architecture — 50+ pre-built, battle-tested React components that AI configures and assembles (not arbitrary code generation). This gives us ~95% first-attempt success rate and sub-5-second generation.

We'd love your feedback. What would you build first?
